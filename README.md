## Установка

#### Клонировать репозиторий

`git clone https://github.com/G28XYZ/php-auth.git`

#### Перейти в каталог

`cd php-auth`

#### Запустить локальный сервер и перейти по адресу - http://localhost:8090

`php -S localhost:8090`
